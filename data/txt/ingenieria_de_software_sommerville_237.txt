Este diagrama sirve para identificar operaciones que se probarán y para ayudar a diseñar los casos de prueba para efectuar las pruebas. Durante las pruebas, se debe crear un resumen de datos que sirva para comprobar que el reporte se organiza correctamente. 2. Una petición de entrada para un reporte a WeatherStation da como resultado la generación de un reporte resumido.
